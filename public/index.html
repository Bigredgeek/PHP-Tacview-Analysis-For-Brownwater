🔧 About

This tool analyzes Tacview mission recordings exported to XML format from DCS World and other combat flight simulators. It provides professional military-style debriefing reports with detailed combat statistics.

Compatible with: DCS World, IL-2 Sturmovik, and other Tacview-supported simulators.

PHP Version: 8.4+ (Modern constructor syntax)
